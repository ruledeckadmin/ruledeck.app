Getting started

Do I need a developer to use Ruledeck?

No. The rule editor runs in any browser and is designed for business users — operations, risk, HR, product. You build decision tables by clicking, not coding. Your IT team handles the one-time setup (Docker install, about 10 minutes). After that, business users own their rules entirely.

What does the setup process look like?

Your IT team needs Docker installed on a server. Then: create a .env file with your credentials, add the provided docker-compose.yml, and run docker compose up -d. Ruledeck starts, validates the licence, and is accessible in the browser. A full IT guide with step-by-step instructions is included with the download.

What server do I need?

Any server running Docker. A €5/month virtual server (2 vCPU, 4GB RAM) handles hundreds of thousands of decisions per month. Ruledeck runs on Linux, macOS, and Windows Server. Many teams run it on an existing server they already have, without provisioning anything new.

I run a small business. Is Ruledeck overkill for me?

Not necessarily. If you’re making the same decision repeatedly — which orders qualify for free shipping, which customers get a discount, how to route incoming requests — and currently doing it manually or through a spreadsheet, Ruledeck automates that. It runs on a €5/month server, takes 10 minutes to set up, and there’s no ongoing subscription. For most businesses making dozens of decisions per day, the time saved pays for itself quickly.

I currently use Excel for business logic. Can I migrate?

Yes. Ruledeck supports Excel import, but the file needs to be in the correct format — not just any spreadsheet. A ready-to-use Excel template is included with your download. Open it, rename the condition and result columns to match your fields, fill in your rule rows, and import. The template includes full instructions and operator reference. A JSON template is also included if you prefer that format.
Licence & pricing

What exactly does the licence cover?

One licence covers one organisation and allows activation on one production instance and one test instance. Both instances validate against the same licence key. The licence is perpetual — you pay once and use Ruledeck indefinitely. No subscriptions, no per-decision fees, no renewal.

Can I run multiple instances?

Each production deployment requires a separate licence. If you need additional production installations — for different teams, regions, or environments — contact us to discuss multi-licence pricing at hello@ruledeck.app.

Can I share my licence key with someone else?

No. Licence keys are non-transferable and issued to the purchasing account only. Sharing a key with third parties, posting it publicly, or using it across unrelated organisations is a breach of the Terms of Service and will result in immediate revocation without refund. Activations are monitored and keys found to be misused will be deactivated.

Is there a refund policy?

Yes. Refunds are available within 14 days of purchase, provided the licence has not been activated. To request a refund, contact hello@ruledeck.app with your order number. Full details are in the Terms of Service.
Technical

How does my system call Ruledeck?

With a standard HTTP POST request. Any language that can make an HTTP request works — Python, JavaScript, PHP, Java, or anything else. You send your input values as JSON, Ruledeck evaluates and returns the result in milliseconds. Full API documentation is included with the download.

What happens when I need an update?

Updates are distributed as new Docker image versions. When an update is available, your IT team runs docker compose pull && docker compose up -d. The licence key carries forward. Your data is in a persistent volume and is never touched by an update.

Do I get the source code?

No. Ruledeck is distributed as a compiled Docker image. You get everything needed to run and operate the software, but not the source code. If your organisation has a specific requirement for source code access, contact us to discuss options.

What’s the difference between Ruledeck and Zapier or n8n?

Zapier and n8n are workflow automation tools — they connect apps and trigger actions. Ruledeck is a decision engine — it evaluates business logic and returns a result via API. They often complement each other: Zapier or n8n can call Ruledeck as part of a workflow to get a decision, then route based on that result.
Data & security

Is my data safe? Does Ruledeck send anything externally?

The only outbound call Ruledeck makes is a licence validation to Lemon Squeezy on startup. No decision data, rule logic, customer information, or usage analytics is ever transmitted externally. Everything runs within your own infrastructure. This makes GDPR and data-residency compliance straightforward.

What if Ruledeck stops being developed or supported?

Your licence is perpetual. You own the version you purchased and it continues to run indefinitely — it does not phone home or expire. If we ever discontinue Ruledeck, existing licences remain valid and the software remains yours to use. This is written into the Terms of Service.

Can business users change rules without involving IT?

Yes, that’s the point. Once IT has deployed Ruledeck, business users log in through their browser, edit rules in the visual editor, test in the sandbox, and publish — all without writing code or opening an IT ticket. Changes go live in seconds. Every change is logged in the audit trail for accountability.
