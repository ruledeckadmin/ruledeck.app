How it works

Three steps.
No developers required.

🗂
STEP 01

Define your logic as a table

Open Ruledeck in any browser. Add condition columns for the inputs your decision depends on — a credit score, a loan amount, a customer segment, a transaction flag. Add result columns for what you want back — a decision, a rate, a routing target, a risk label.

Each row is one rule. The engine evaluates top to bottom and returns the first match — or all matches if you need them. Add an ELSE row as a fallback. No formulas, no code, no IT ticket.

🚀
STEP 02

Test, then publish with one click

Before going live, use the built-in testbench to run your rule against real inputs. Debug mode shows exactly which row matched and why. When you're satisfied, press Publish.

Ruledeck instantly creates a live REST API endpoint. Your IT team connects it once. Every published version is saved with a timestamp — roll back to any prior version in seconds if something goes wrong.

✏️
STEP 03

Iterate without engineering

Policy changed overnight? New product tier to add? Fraud threshold to tighten? Open the rule in your test environment, edit the table, verify in the testbench, and publish. The update is live in seconds.

Export to Excel, share with a colleague for sign-off, import back in. Every change is logged in the audit trail with the user identity and timestamp — suitable for compliance evidence.

Data & Security

Your data
stays yours.

Unlike SaaS decision platforms, Ruledeck runs entirely within your own environment. No decision data, customer information, or business logic is ever sent to an external server.

  • 🖥

    Self-hosted deployment

    Install on your own server, data centre, or private cloud. You control the infrastructure and the data.

  • 🔑

    API key + session authentication

    Solve endpoints require a valid API key. The admin UI requires session authentication with brute-force lockout after 5 failed attempts.

  • 🛡️

    Hardened by default

    Helmet security headers, HttpOnly/SameSite cookies, rate limiting, body size enforcement, and SQL injection protection ship enabled out of the box.

  • 📡

    No telemetry, no phone-home

    Ruledeck does not contact any external server after installation. No usage data, no analytics, no licence calls in the critical path.

  • 🌍

    GDPR / data sovereignty

    Data never leaves your servers, making regional data-residency compliance straightforward for regulated industries.
